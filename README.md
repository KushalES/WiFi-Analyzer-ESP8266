# 📡 Wi-Fi Analyzer using ESP8266

## 🔍 Overview

This project is a standalone Wi-Fi analyzer built using ESP8266. It scans nearby networks and displays useful information like signal strength, channel usage, and security type through a web dashboard.

## 🚀 Features

* 📡 Scan nearby Wi-Fi networks
* 📊 Signal strength visualization (RSSI graph)
* 📡 Channel analysis & recommendation
* 🔐 Security detection (WPA/WPA2/Open)
* 🌐 Web-based dashboard
* 📱 Standalone Access Point mode

## ⚙️ Technologies Used

* ESP8266 NodeMCU
* Arduino IDE
* Embedded C++
* Wi-Fi (IEEE 802.11)

## 📱 How to Use

1. Upload code to ESP8266
2. Connect to Wi-Fi:

   * SSID: WiFi-Analyzer
   * Password: 12345678
3. Open browser:
   http://192.168.4.1
